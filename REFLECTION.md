The challenging part of the project for me was keeping up with the appropriate imports and exports for each file. I have struggled with the file structure when designing apps, so I leaned on Google Gemini to help design the layout of necessary files.  In components, it was recommended that I create and utilize a Spinner file to improve performance and prevent the app from stalling and breaking when requests were made from the API.
The Spinner was also the design decision that was made to improve the reliability of the app's functions.  I didn't want the user experience to be flawed and frustrating while waiting on components to respond.
